# LIFE IA - Site Portfolio

Site institucional da aplicação **LIFE IA - Regulação Autônoma**, desenvolvida para otimizar a regulação hospitalar no SUS-Goiás através de inteligência artificial.

## Sobre

Este repositório contém o site portfolio que apresenta a solução LIFE IA, incluindo:

- Visão geral do problema da regulação hospitalar
- Pipeline inteligente (Ingestão, Inteligência, Síntese)
- Lógica de peneira para escolha de hospitais
- Áreas de operação (Dashboard, Hospitalar, Regulação, Transferência, Auditoria)
- Diferenciais estratégicos e stack tecnológico

## Deploy

O site é hospedado via GitHub Pages a partir da pasta `docs/`.

**URL:** [lifeia.tec.br](https://lifeia.tec.br)

Para gerar um novo build:

```bash
pnpm install
pnpm run deploy:ghpages
```

## Aplicação principal

O código-fonte da aplicação de regulação autônoma está em: [github.com/LiviaMor/regulacao-ms](https://github.com/LiviaMor/regulacao-ms)
